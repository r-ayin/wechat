# 神级深度财经研究 - GODTIER Deep Research

使用13层分析架构（含L-1残差Alpha层）、105+顶级信源、脚本计算强制层和三层幻觉检测，生成具有认知穿透力的深度财经分析文章。

## 技能概述

**核心价值**：
- **13层分析架构** - 从残差Alpha扫描到预测追踪的完整情报分析链
- **105+顶级信源** - URL锚定，每个数据点可追溯到具体网页
- **脚本计算强制层** - 所有数字运算由Python脚本执行，消除LLM幻觉
- **三层幻觉检测** - 数字级+逻辑级+全文级主动检测
- **多Agent并行** - 效率提升8-10倍，2小时完成神级深度研究

**适用场景**：
- 公司深度分析（上市公司/未上市独角兽）
- 行业结构性变革研究
- 投资策略与风险评估
- 宏观趋势与政策影响分析
- 竞争格局与博弈分析
- 技术颠覆与商业化路径
## 分析框架：13层情报分析架构

### 层级结构与核心问题

```
Level -1: 残差Alpha    -> Alpha在误差中：残差扫描/共识误判/尾部误定价/信息延迟
Level 0:  反向证伪     -> 我能怎么错？先证明自己可能错
Level 1:  范式谬误     -> 共识错在哪？击碎共识制造智力落差
Level 2:  系统约束     -> 硬约束是什么？WACC/监管/物理/时间
Level 3:  激励机制     -> 为什么他们别无选择？管理层/投资者/监管者
Level 4:  信息博弈     -> 谁知道什么？内部人/信号/逆向选择
Level 5:  微观操作     -> 原子动作是什么？单位经济/护城河工程化
Level 6:  对手盘+网络  -> 谁赚钱谁亏钱？级联风险/流动性黑洞
Level 7:  时间动态     -> 反馈循环和临界点？阈值/相变/自反性
Level 8:  行为心理     -> 人会怎么犯错？认知偏差/叙事/羊群
Level 9:  定量模型     -> 数学说什么？脚本执行DCF/蒙特卡洛/回归
Level 10: 终局推演     -> 三情景+黑天鹅+观察指标
Level 11: 预测追踪     -> 我上次说对了吗？闭环校准
```
### 六维评价体系

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 残差Alpha识别 | 15% | 5分:系统性识别误差中的具体Alpha机会+量化估算; 4分:识别误差方向但未量化; 3分:提到残差但无系统分析; 2分:仅提及概念; 1分:完全缺失 |
| 认知穿透力 | 20% | 5分:揭示结构性约束+"原来他们别无选择"; 4分:揭示约束但不够震撼; 3分:有深度但停留在描述; 2分:表面分析; 1分:常识性内容 |
| 逻辑闭环 | 20% | 5分:机械性传导链条+每个推论可证伪+有反面证据; 4分:链条完整但缺反面证据; 3分:主要推论有支撑但有跳跃; 2分:多个逻辑跳跃; 1分:断言式结论 |
| 颗粒度 | 15% | 5分:精确到数字+时间+人物+操作细节; 4分:有数据但缺操作细节; 3分:有数据但不够微观; 2分:笼统数据; 1分:无具体数据 |
| 宏观微观折叠 | 15% | 5分:完整"美联储→资本成本→公司决策→个人KPI"传导链; 4分:有传导但缺某一层; 3分:宏观微观都有但未折叠; 2分:只有宏观或微观; 1分:无传导 |
| 可操作性 | 15% | 5分:明确行动指令+具体观察指标+阈值+数据源; 4分:有指令但阈值模糊; 3分:方向性建议; 2分:笼统建议; 1分:无可操作内容 |

**评分校准**：
- 90-100: 投行顶级研报水准（Goldman/Morgan Stanley核心研报）
- 80-89: 优质对冲基金内部研究（Citadel/DE Shaw内部备忘录）
- 70-79: Doomberg最佳文章水准
- 60-69: 普通券商研报
- <60: 不达标，需要重写

**触发重写的最低分**：
- 残差Alpha识别 <10/15, 认知穿透力 <14/20, 逻辑闭环 <14/20
- 颗粒度 <10/15, 宏观微观折叠 <10/15, 可操作性 <10/15
