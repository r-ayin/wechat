# ⚖️ wechat 项目协议 (PROTOCOL)

> **性质**：本项目宪法。独立管线不依赖外部工作区。
> 本文件是精简版，仅保留项目级规则。完整版见 x-tool 工作区。

---

## §1 项目文件三件套（强制）

| 文件 | 作用 |
|------|------|
| `CLAUDE.md` | 项目操作规范（自动加载） |
| `PROGRESS.md` | 进度追踪 |
| `GATES.md` | 质量门禁清单 |

---

## §2 变更三阶段

```
① BEFORE → ② CHANGE → ③ AFTER
  读取规范     执行变更     更新进度
```

---

## §3 PROGRESS.md 格式

- 每次变更后更新 `> 最后更新：` 时间戳
- 已完成项用 `[x]` 标记
- 当前任务写清楚阻塞原因

---

## §4 GATES.md 门禁

| 级别 | 含义 |
|------|------|
| 🔴 CRITICAL | 不通过不得产出 |
| 🟡 IMPORTANT | 不通过需注释原因 |
| 🟢 NICE | 尽量满足 |

---

## §5 Agent 行为准则

1. 进入项目目录 → 先读 CLAUDE.md
2. 变更后 → 更新 PROGRESS.md
3. 产出前 → 检查 GATES.md
4. 禁止删除用户文件
5. 禁止绕过门禁
