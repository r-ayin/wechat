## 对标写法分析：为什么你的文章"太虚"

### ❌ 你的文章的问题

```
开头: "现代人还需要心理治疗吗？" ← 抽象问题，没有具体的人
中间: 拉康说...马克思说...福柯说... ← 理论堆叠，没有故事
结尾: "让我们一起..." ← 空洞的号召
```

### ✅ 对标账号的写法

```
开头: "李晴在东莞工厂做了八年心理咨询师，她发现..." ← 一个具体的人
中间: 李晴遇到的三个真实案例 → 引出"为什么工厂制造抑郁" → 引用数据支撑
结尾: "李晴还在工厂。她说..." ← 回到这个人的真实处境
```

### 改写规则

**铁律：每篇文章必须有至少——**
1. 一个有名有姓的真实人物
2. 一处具体的场景描写（时间+地点+环境）
3. 一组可验证的数据或事实
4. 一个当前正在发生的热点事件作为钩子
5. 理论引用不超过全文20%，且必须服务于解释具体现象

## 对标文章的格式模板

从真实故事计划/人物/GQ报道提取的标准结构：

```
标题: [具体的人/具体的处境] + [普遍性问题暗示]
例: "我在东莞工厂做了八年心理咨询师，见证了抑郁如何被制造"

开头(200字): 
一个具体场景。有画面、有声音、有情绪。不抛理论。

第一部分(500字):
展开这个人的故事。发生了什么。具体细节。

第二部分(500字):
从这个人的故事拉远——背后是什么结构性问题？
引用数据、研究、政策。理论在这里承重，不露骨。

第三部分(500字):
回到具体。这个人怎么应对的？其他类似处境的人呢？
呈现矛盾——不给出简单答案。

结尾(200字):
回到开头的场景或人物。给读者留一个问题，不给答案。
```
