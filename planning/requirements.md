# wechat — 需求文档

> 阶段：① 需求 | 2026-06-26 | 微信公众号深度长文项目（从 douyin 拆分独立）

## 项目愿景

「被压迫者小组」— 信息差平权者。用深度研究内容帮普通人磨平 AI 时代的信息差。
最终输出形态：微信公众号深度长文（15000-30000 字/篇）。

## 目标受众

对社会/劳动/教育/心理/技术权力有切身体感、但缺乏系统分析工具的普通人。

## 当前状态：🟢 活跃生产

```
✅ godtier-deep-research（双模式 finance + general，13 层分析）
✅ persona 人格三件套（SOUL + STYLE + PERSONA，全量注入）
✅ QA 四步门禁（搜证 → 逻辑审查 → L4 防幻觉 → 迭代修复）
✅ 双模式入口（🚀自动 / 🎯手动 Dankoe 采访）
✅ hot-scanner v2.0（43 关键词×五支柱 + 事件触发 + 五维评分）
✅ 15 篇微信长文已产出（11 hot + 4 evergreen）
✅ 对标账号数据库 + 常青选题池
```

## 核心用户

| 用户类型 | 需求 | 满足度 |
|---------|------|--------|
| 结构性议题关心者 | 想看透现象背后的系统逻辑 | 🟢 管线就绪 |
| 信息饥渴者 | 想要深度但有温度、有人格的内容 | 🟢 persona 注入 |
| 热点跟进者 | 想要事件触发后的深度解读（非快讯） | 🟢 hot 分支 |

## 内容管线（六阶段强制）

```
选题(热点/常青/对标) → ①竞品蒸馏 → ②研究简报 → ③godtier研究 → ④persona重写 → ⑤QA门禁 → ⑥输出
（pipeline-gate.sh 每阶段 verify，不通过不可进下一阶段）
```
