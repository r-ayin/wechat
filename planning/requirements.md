# douyin — 需求文档

> 阶段：① 需求 | 2026-06-20 | 方向已确认，进入建设

## 项目愿景

"信息差平权者" — 用深度研究内容帮普通人磨平 AI 时代的信息差，让生活变得更好。

## 目标受众

对 AI/科技/社会变化有感知但缺乏系统理解工具的普通人。

## 当前状态：🟢 内容引擎就绪，从未发布

```
✅ godtier-deep-research v2.0（双模式：finance + general）
✅ 拆分引擎 v1.0（短视频 60s + 中视频 3-5min 脚本）
✅ 热点选题入口 v1.0（手动→管线编排）
✅ MD→双HTML 转换器（文章版 + 分析结构版）
✅ HTML→PNG 长图渲染器
✅ 微信推送闭环（iLink 长图 + 双HTML）
✅ EvoMap 深度研究（首篇完成：10930字·87/100·85+信源）
✅ 竞品认知修正（14+ 同类工具发现）
❌ 从未发布过任何内容到抖音
❌ 无粉丝、无数据反馈
❌ 无内容日历
```

## 核心用户

| 用户类型 | 需求 | 当前满足度 |
|---------|------|-----------|
| 科技焦虑者 | 想知道 AI 对自己工作生活的影响 | 🟡 引擎就绪 |
| 信息饥渴者 | 想要深度但有温度的内容 | 🟡 引擎就绪 |
| 普通刷到的 | "这个讲得挺明白" → 关注 | 🔴 无内容 |

## 待实现需求

- [ ] **P0-1 选题确认**：从热点选题入口选第一个主题

- [ ] **P1-1 内容日历**：每周 2-3 更的计划

- [ ] **P1-3 数据反馈闭环**：播放/互动数据 → 优化选题方向

### 🟢 P2 — 多平台分发
- [ ] **P2-1 小红书同步**：深度内容的小红书版本

## 内容管线（已设计，待验证）

```
热点选题 → godtier 深度研究 → 拆分引擎 → 短视频脚本 → 用户拍摄
                                                        ↑
                                              引擎边界（引擎不出视频）
```

## 关键决策点

> ⚠️ douyin 的瓶颈不在引擎，在人。引擎能产出脚本，但需要人拍+剪+讲。
> 如果用户还没有开始拍摄，是否先：
> 1. **改走图文路线**：内容先发小红书（wanxia 管线可复用）
> 2. **纯音频+PPT**：降低拍摄门槛
> 3. **外包剪辑**：脚本 → 给剪辑师
