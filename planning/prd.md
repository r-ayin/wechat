## 内容规格

### 短视频（60s，抖音主战场）

| 要素 | 规格 |
|------|------|
| 时长 | 45-60s |
| 结构 | 钩子(3s) → 冲突(15s) → 分析(30s) → 总结(10s) |
| 钩子 | 拆分引擎 7 种：数字/疑问/反常识/痛点/故事/对比/结果 |
| 视觉 | 口播 + 信息图/数据卡叠加 |
| 互动 | 结尾提问引导评论 |

### 中视频（3-5min，B站/视频号）

| 要素 | 规格 |
|------|------|
| 时长 | 3-5min |
| 结构 | 引入→背景→深度分析→多方观点→个人判断 |
| 深度 | 比短视频多 3x 信息量 |
